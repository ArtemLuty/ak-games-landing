# AK Games

Minimal premium landing page for the AK Games indie mobile apps and games brand.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The project is configured with `output: "export"` so the static site is generated into the `out/` directory and can be deployed easily to static hosting providers.
